# ⚙ Gear Machine 

gear machine é um programa que inicialmente foi criado para a turma de mecanica do ifpa, tendo como sua função calcular engrenagens de dentes retos, conica e parafuso sem fim.

## 🔹 Tecnologias Utilizadas 
- Html
- Css
- JavaScript
- Jquery
- Figma
